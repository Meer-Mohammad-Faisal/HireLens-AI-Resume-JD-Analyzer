#  HireLens AI (RoleMatch AI) – Resume JD Analyzer

HireLens AI is an AI-powered Resume & Job Description Analyzer that helps candidates optimize their resumes for specific job roles using intelligent matching and AI-driven insights.

🔗 Live Demo: https://hire-lens-ai-resume-jd-analyzer.vercel.app/

---

##  Features

-  Upload Resume (PDF/DOC)
-  AI-Powered Resume Analysis
-  Match Resume with Job Description
-  ATS Score & Skill Gap Detection
-  Personalized Improvement Suggestions
-  Resume Download Support
-  Secure Authentication & User Management
-  Fast & Responsive UI

---

##  Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### AI & APIs
- Gemini/OpenAI API
- Resume Parsing
- NLP-based JD Matching

### Deployment
- Vercel (Frontend)
- Render (Backend)

##  How It Works

1. Upload your resume
2. Paste the Job Description
3. AI analyzes resume relevance
4. Get:
   - ATS Match Score
   - Missing Skills
   - Resume Improvement Suggestions
   - Overall Compatibility Analysis

##  Installation

### Clone the repository

```bash
git clone https://github.com/Meer-Mohammad-Faisal/HireLens-AI-Resume-JD-Analyzer.git
```

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd Backend
npm install
npm start
```

---

##  Environment Variables

Create a `.env` file in backend:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_api_key
```
